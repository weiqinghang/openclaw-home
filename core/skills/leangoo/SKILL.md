# Leangoo Skill

管理 Leangoo 看板上的用户故事卡片。

## 快速开始

### 打开看板

- 项目页面：`https://www.lg.team/kanban/project/go/{project_id}`
- 看板页面：`https://www.lg.team/kanban/board/go/{board_id}`

### 操作流程

1. **打开浏览器** → `browser` 工具
2. **导航到看板** → `navigate` 操作
3. **执行操作** → 创建/编辑/移动卡片

---

## 卡片操作

### 1. 创建卡片

1. 点击列表下方「添加卡片」
2. 输入卡片标题
3. 按 `Enter` 提交

**提示**：可用下拉选择卡片类型（任务/用户故事/缺陷）

### 2. 编辑描述

1. 点击卡片打开详情弹窗
2. 点击「描述·编辑」
3. 输入 Markdown 内容
4. 点击「保存」

**支持**：Markdown、图片粘贴 `Ctrl+V`、复选框 `- [ ]`

### 3. 检查项

1. 点击「检查项」按钮
2. 输入检查项，Enter 提交
3. 勾选/取消复选框

### 4. 移动卡片

**方法 A**：详情弹窗 → 「移动」→ 选择目标看板/列表

**方法 B**（推荐）：直接拖拽卡片到目标列表

### 5. 标签

1. 点击「标签」按钮
2. 选择已有标签 或 新建标签
3. 点击标签添加/取消

### 6. 评论

1. 在评论框输入文字
2. `Ctrl+Enter` 保存
3. 支持 @提及成员

### 7. 时间

点击「开始时间/截止时间/完成时间」→ 选择日期时间 → 确定

### 8. 附件

点击「附件」→ 上传文件 或 添加链接

### 9. 其他操作

| 功能 | 说明 |
|------|------|
| 成员 | 分配负责人 |
| 工作量 | 设置预估工时 |
| 关联 | 关联其他卡片 |
| 复制 | 复制卡片 |
| 删除 | 删除卡片 |

## 看板配置

### 看板菜单（右上角）

- 设置周期、Sprint 时间
- 设置所属项目
- 可见性（公开/私有/项目成员）
- 添加卡片类型
- 看板统计
- 导入/导出
- 归档

---

## 列表配置

### 列表菜单（列表右上角）

- 添加卡片
- 复制列表
- 创建引用
列
- 删除列表

### 添加列表/泳道

看板底部：「- 设为完成添加列表」或「添加泳道」

---

## 创建新看板

1. 进入项目页面
2. 展开项目，点击「新建看板」
3. 输入名称，选择模板
4. 创建后添加列表和卡片

---

## 注意事项

1. 需登录 Leangoo 账号
2. 确保有目标项目权限
3. 操作前等待页面加载
4. 快捷键：
   - 添加卡片: Enter
   - 描述/评论: Ctrl+Enter
   - 检查项: Enter
