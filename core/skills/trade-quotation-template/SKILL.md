# Trade Quotation Template

## 目标

围绕既有报价模板做 4 件事：

1. 读取模板字段与版式
2. 从 SKU 主数据填充报价项
3. 将产品图放到 `PHOTO` 列对应位置
4. 导出前校验金额、条款、图片和打印区

## 当前模板事实

- 模板主体为单工作表报价单
- 打印区：`A1:P29`
- 页面方向：纵向
- 主要商品区：`11:17` 行
- `B11:B17` 为图片位
- `A18:P29` 为汇总、条款、银行信息

## 输入

- 客户抬头信息
- 选中的 SKU 列表
- 每个 SKU 的数量、单价、包装参数
- 本地图路径

## 输出

- 报价 Excel
- 如环境支持，再导出 PDF

## 生成规则

1. 只在既有模板内填值，不重新发明版式
2. 图片必须落到 `PHOTO` 列对应商品行
3. 总价、大写金额、付款条款必须同步更新
4. 若缺图片、尺寸、包装参数，先报缺口，不盲填

## 推荐数据源

- SKU 主数据
- 本地图库
- 客户主数据

## 当前本地约定

- 图片默认放太白工作区
- 报价模板与生成结果也放太白工作区
