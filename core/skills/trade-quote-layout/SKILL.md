# Trade Quote Layout

适用于“报价单必须插产品图片，且排版和尺寸要求很强”的场景。

## 目标

生成一个**可直接交付**的报价单版式方案，而不是只有字段列表。

## 推荐交付介质

按优先级：

1. Excel / 表格版报价单
2. 飞书表格
3. 飞书文档封面说明 + 表格主体
4. 如果客户要正式版，再导出 PDF

## 图片型报价单的关键做法

1. 先定版式，不先填内容
2. 先定图片框尺寸，再收图
3. 先定列宽、行高、纸张尺寸、页边距
4. 再批量填产品信息

## 必备规格信息

没有这些信息，排版很难稳定：

- 页面尺寸
  - A4 / Letter / 横版 / 竖版
- 页边距
- 每页产品数量
- 每个产品卡片的宽高
- 图片框尺寸
- 图片裁切规则
  - 等比缩放
  - 铺满裁切
  - 留白适配
- 字体
- 字号层级
- logo 位置
- 公司抬头信息
- 报价币种
- 小计/总计区块
- 备注区块
- 页眉页脚

## 产品图片要求

建议提前储备：

- 主图原图
- 白底图
- 尺寸图
- 细节图
- 图片命名规则
- 最低分辨率
- 宽高比
- 背景规范

推荐至少统一：

- 长边 >= 1200 px
- 背景统一
- 产品主体居中
- 命名可映射 SKU

## 建议储备的信息

### 公司级

- 公司英文名 / 中文名
- logo
- 地址
- 联系方式
- 默认贸易条款
- 默认付款条款
- 默认交期说明

### 产品级

- SKU
- 产品名称
- 英文名称
- 规格
- 材质
- 包装方式
- MOQ
- 单价
- 图片路径
- 备注

### 模板级

- 报价单模板版本
- 目标客户类型
- 使用语言
- 报价币种
- 页面风格
- 一页放几项产品

## 输出时必须给的内容

1. 页面规格
2. 表格结构
3. 图片摆放规则
4. 字段顺序
5. 缺失信息清单
6. 可直接执行的制作步骤

## 默认建议

如果用户没给排版规范，默认：

- A4 纵向
- 首页放公司信息 + 报价说明
- 内页每页 4 个产品
- 每个产品 1 张主图
- 图片框固定尺寸
- 价格区、规格区、备注区分层显示

## 风险提醒

- 图片来源不统一，会直接毁掉报价单观感
- 没有固定图片框尺寸，后期会反复返工
- 没有 SKU 命名规则，批量换图会失控
- 没有模板版本，后续维护会混乱
