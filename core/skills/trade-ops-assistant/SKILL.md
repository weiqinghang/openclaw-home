# Trade Ops Assistant

适用于太白金星的日常外贸执行工作。

## 目标

把外贸工作从“给建议”推进到“出分析、出表格、出跟进结果”。

## 适用任务

- 电商平台选品分析
- 竞品与价格带分析
- 供应商询价与比价
- 报价单制作
- 订单全流程跟进
- 阶段汇报整理

## 推荐工具组合

1. `web_search` / `web_fetch`
2. 浏览器
3. 表格能力
4. 飞书文档 / 飞书表格

## 工作流程

1. 先明确业务阶段：
   - 选品
   - 询价
   - 报价
   - 下单
   - 跟单
   - 出货
   - 回款
2. 再明确交付物：
   - 分析表
   - 报价单
   - 跟进表
   - 汇报文档
3. 输出必须结构化：
   - 字段清单
   - 表格列设计
   - 状态字段
   - 风险字段
4. 每轮推进后补一段汇报：
   - 已完成
   - 待确认
   - 风险点
   - 下一步

## 默认输出结构

### 选品分析

- 平台
- 品类
- 目标市场
- 价格带
- 竞品数量
- 头部卖点
- 供应链难度
- 毛利判断
- 风险备注

### 报价单

- SKU
- 产品名
- 规格
- 图片
- MOQ
- 单价
- 币种
- 贸易条款
- 打样费
- 包装
- 交期
- 付款方式
- 备注

### 订单跟进

- 客户
- 产品
- 当前阶段
- 最近动作
- 下一动作
- 截止时间
- 风险等级
- 责任人

## 原则

- 不只给建议，要给可执行表格结构
- 不只写结论，要给判断依据
- 不只说“跟进”，要明确下一步动作和时间点
